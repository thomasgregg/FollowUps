Common troubleshooting

  • Confirm your OpenAI API key is set in FollowUps Settings.
  • Allow microphone access in iOS Settings for FollowUps.
  • Allow notifications if you want completion alerts while processing in background.
  • If reminders do not sync, check Reminders permission in iOS Settings.
