## Guidelines 5.1.1(i) and 5.1.2(i) - Privacy, Data Collection, Data Use

FollowUps sends user data to OpenAI only for core app functionality (transcription and task extraction), and only after explicit user action and consent.

- **What data is sent:** recording audio and transcript text from the recorded session.
- **Who receives it:** OpenAI (API provider).
- **When it is sent:** only after the user starts recording, stops recording, and allows cloud processing.
- **Permission controls:** the app asks for microphone permission, and also requires explicit in-app permission before cloud processing is used. Users can disable cloud processing in Settings at any time.
